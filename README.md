## 📖 简介 (Introduction)

**Office Oasis** 是一款专为职场人设计的“合法摸鱼”应用。它基于 **组织行为学** 中的“资源保存理论 (COR)”，旨在将碎片化的低效摸鱼时间，转化为高质量的 **微休息 (Micro-breaks)**。

不同于普通的娱乐应用，它设有严格的 **15分钟倒计时**，时间一到强制打烊，帮助用户建立健康的休息边界。

## ✨ 核心模块 (Modules)

### 1. 💥 情绪粉碎机 (Stress Shredder)
* **痛点：** 积压的负面情绪无处宣泄。
* **玩法：** 极简交互，点击“一键消除班味”，配合解压的撒花特效与心理宽慰语录，物理超度烦恼。
* **隐私：** 无需输入任何文字，零风险释放压力。

### 2. ☕ 知识特调饮 (Knowledge Brew)
* **痛点：** 摸鱼时的空虚感与负罪感。
* **玩法：** 模拟咖啡冲泡过程（1.5秒仪式感动画），随机掉落 Excel 技巧、沟通话术或心理学冷知识。
* **价值：** 让休息不仅是放松，更是微学习。

### 3. 🍾 同事漂流瓶 (Office Bottle)
* **痛点：** 职场孤独感与信息闭塞。
* **玩法：** 强制匿名的留言板。在这里，你可以安全地扔出吐槽，或者捞起某个陌生同事的心声。

## 🎨 设计理念 (Design Philosophy)

* **疗愈系 UI:** 采用抹茶绿 (#6B8E23) 与 米色 (#FDFCF8) 的配色，配合圆润的卡片设计，降低视觉噪点，平复焦虑。
* **响应式布局:** 完美适配 PC 宽屏与手机竖屏，随时随地由你掌控。
* **纯净架构:** 无后端、无数据库、无追踪，最大程度保护职场隐私。
